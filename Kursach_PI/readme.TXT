Для работы данного проекта необходимо следующие:

1) MySQL Server от версии 5.5

2) Любой веб сервер по типу Apach 2.4 с возможностью работы PHP.

3) Необходимо запустить скрипт Kursach_HospitalBD.sql для этого можно использовать MySQL WorkBench

4) При регистрации необходимо,чтобы первые 6 пользователей были врачами у которых логин это их фамилия(Lavrov - логин)
Для назначение привелегии необходимо открыть sql скрипт testbd.sql и запустить его.
